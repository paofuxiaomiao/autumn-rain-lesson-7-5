# 《秋天的雨》交互课件 · 设计理念（Ideas）

## 一、设计哲学：山水咨询主义 × 秋日绘本

本课件的设计哲学继承「山水咨询主义（Landscape Consulting）」——McKinsey 式咨询版式骨架与中国山水雾面自然气质的融合，并针对小学三年级语文课堂做了「秋日绘本化」的转译。四大信条不变：

> **留白即结构、自然即隐喻、网格即纪律、一页一论点。**

在课件语境下，这四条被具体化为：每页只讲一个自然段、一个论点（一页一论点）；大面积暖雾白底与克制的插画占位让画面「呼吸」（留白即结构）；秋雨、落叶、雨滴既是氛围动效也是课文意象本身（自然即隐喻）；页眉—标题—正文—解读—插画的固定网格贯穿全部 11 页（网格即纪律）。

## 二、为什么是「衬线 + 留白 + 低饱和」

课文《秋天的雨》是一篇抒情散文，语言温柔、意象丰富。设计上避免常见儿童课件的高饱和卡通风，而选择：

1. **衬线字体（Noto Serif SC）作为课文正文与标题**——衬线的笔触感接近课本铅字与书法教育语境，帮助学生建立「正式阅读」的仪式感；行高 2.1、字距 0.04em 的课文排版接近教材版式，便于逐字指读。
2. **暖雾白底（#FAFAF7）+ 大量负空间**——减少视觉噪音，让注意力落在课文与插画上；每页至少 40% 留白。
3. **低饱和秋日色板**——银杏黄、枫叶红、柿橙都压低饱和度，唯一的高显色「柿橙强调色」只用于关键词、交互反馈与品牌粒子，出现即代表「值得注意」。

## 三、教学目标驱动的页面编排

页面顺序并非线性照搬课文，而是「精读—游戏—精读」交替，符合三年级学生注意力节奏：

| 页序 | 内容 | 教学意图 |
|---|---|---|
| 1 | 封面（张力页） | 以全幅山水插画与课文首句营造氛围，「推开秋天的大门」按钮即课文比喻的交互化 |
| 2 | 第1段 · 钥匙 | 认识总起句与比喻 |
| 3 | 第2段 · 颜料（跟读） | 核心段落：逐句高亮跟读 + 背诵挖空，直接服务课后习题一 |
| 4 | 游戏① 颜色配对 | 即时巩固第2段「五彩缤纷」的内容记忆 |
| 5 | 第3段 · 气味 | 体会拟人（躲、勾住） |
| 6 | 第4段 · 小喇叭 | 认识过冬常识与拟人 |
| 7 | 游戏② 过冬连线 | 巩固第4段动植物过冬方式 |
| 8 | 第5段 · 歌（总结） | 居中仪式化排版，点明「丰收与欢乐」的课文内涵 |
| 9 | 生字学习 | 会认12字+会写12字（田字格），AI 讲解组词 |
| 10 | 课后习题 | 教材三道原题：背诵引导 / 结构梳理 / 仿写+AI点评 |
| 11 | AI 问答 | 「雨滴老师」自由问答，兜底所有个性化疑问 |

## 四、AI 的角色设定：雨滴老师

AI 不是搜索引擎，而是一位有人格的「雨滴老师」：温柔、耐心、只围绕课文与语文学习；回答短（≤150字）、多打比方、**先提示再引导、结尾留启发性问题**——这与三年级「引导发现式」教学法一致。仿写点评遵循「具体表扬 → 检查句式 → 温柔建议」三步结构。

## 五、动效观：动效即课文

所有动效都能在课文里找到出处：翻页的「雾散开」对应秋雨的朦胧；飘落的银杏叶与枫叶对应第2段；细雨粒子对应全文意象；配对成功的轻微弹跳对应「你挤我碰」的欢快。禁止与课文无关的弹跳、旋转、闪烁——动效服务于沉浸，而非炫技。
